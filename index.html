Learn the Basics?

Vaulture gives you one official run each day. Try a quick practice tutorial first so your first real run counts.

How to Play

The Daily Run
Everyone cracks the same vault each day. You get one official scored run.
Stage a Play
Place up to 5 crew in the slots. Your loot is every card's $ value added up. Your multiplier comes only from your biggest matching set — 2 alike ×2, 3 ×4, 4 ×8, 5 ×15.
Read the Receipt
The receipt reads loot × multiplier = take, live before you commit. Off-crew cards add loot but never the multiplier — so matching usually beats mixing.
Commit Carefully
Press Commit and the hand counts for the run. Staged crew only come back out via 🔥 Burn. Crack each lock's target before your plays run out.
Share Your Result
When the run ends, share a spoiler-free result card and compare scores.
